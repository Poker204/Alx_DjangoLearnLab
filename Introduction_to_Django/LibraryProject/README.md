# LibraryProject

This is the initial Django project for the Introduction to Django task.

## How to Run

- Navigate into the LibraryProject directory
- Run the server with python manage.py runserver
- Visit http://127.0.0.1:8000 to see the Django welcome page

## Files of Interest

- manage.py: Django project management tool
- settings.py: Project settings
- urls.py: URL configuration
